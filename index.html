MLClient for Android Documentation¶

Machine Learning Client for Android is a front end for the Machine Learning Layer in GIoTTO stack.

With this client, users without technical background can create their own machine-learning-based sensors, which we call virtual sensors. To use this client, you need Building Depot and Machine Learning Layer property installed and setup.
